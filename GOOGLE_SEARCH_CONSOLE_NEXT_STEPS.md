## 🔧 Troubleshooting

### If pages aren't indexing after 1 week:
1. Check **Coverage** report for errors
2. Use **URL Inspection** to see what Google sees
3. Make sure your site is accessible (no password protection)
4. Verify robots.txt allows crawling

### If you see errors:
- **"Submitted URL blocked by robots.txt"**: Check your robots.txt file
- **"Page not found"**: Make sure the URL is correct
- **"Crawled but not indexed"**: This is normal, Google will index it eventually
